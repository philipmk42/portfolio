# Philip Mathew | Portfolio

A modern, responsive personal portfolio website showcasing my work as an AI/ML Engineer specializing in Computer Vision, NLP, and full-stack development.

## Live Site

🔗 [View Portfolio](https://philipmk42.github.io/portfolio)

## Features

- Clean, modern dark-themed design
- Responsive layout with mobile navigation
- Smooth scroll navigation
- Interactive project showcase
- Education & experience timeline
- Skills & technologies section
- Contact section with direct links

## Tech Stack

- **HTML5** + **CSS3** + **JavaScript**
- Custom fonts (Bricolage Grotesque, Hanken Grotesk, JetBrains Mono)
- Intersection Observer for scroll animations
- Fully responsive design

## Sections

- **Hero** — Introduction and call-to-action
- **About** — Background, education, and focus areas
- **Work** — Selected projects with descriptions and technologies
- **Skills** — Technical skills across ML, Web, and Programming
- **Path** — Education and professional timeline
- **Contact** — Ways to reach out
